# UI Wireframe Notes (Implementation Guidance)

## Top-level layout
- Left sidebar: section list
- Right main: existing toolbar + scroll content
- Sticky toolbar recommended

## Sidebar content (minimum)
- Header: “Sections”
- Section list:
  - Title
  - Optional badge (missing count) or checkmark

## Status row in toolbar
- File: <basename>
- Dirty indicator
- Last saved

## Interaction defaults
- Smooth scroll on nav click
- Collapse on section header click
- Persist collapse state by quote type
